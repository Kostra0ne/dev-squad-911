What is D.O.M

D.O.M is an A.P.I.
It stands for Document Object Model.
API stands for Application Programming Interface.
There are many API, doing various stuff, giving various services.

The DOM API is focused on HTML documents

The goal of the DOM API is to provide a set of methods, objects and properties, allowing you as a developer, to control your documents the way you want.
