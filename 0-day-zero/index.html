Prerequisites

open a terminal window (default shell, i-term, VSCode terminal etc.)
Then check that :

A / VS Studio Code is installed

B / nodejs is installed

$ node --version

C / Mongodb and mongo Compass are installed
please refer to the official documentation for your current O.S (operating system).
installation doc - get compass

once done...

Intro challenge

once done...
install these two packages globally:

$ [sudo] npm install -g lite-server nodemon

1 - check your location

$ pwd
$ ls

2 - Create a dedicated folder for the bootcamp

$ mkdir Users/you/code/ironhack-bootcamp

This is now your main working directory for the next 2 months.

3 - navigate to the folder with your terminal

$ cd path/to/you/code/ironhack-bootcamp
$ mkdir 0-day
$ cd 0-day
$ touch index.html index.js index.css
$ code .

4 - Add some basic content

> Build a simple HTML file

 <!DOCTYPE html> <html> <head> <title>Ironhack |Â web dev bootcamp - day 0</title> <link rel="stylesheet" href="./index.css"> </head> <body> <h1>Hello, World</h1> <p>foo, bar, baz</p> <p>please check the <a href="https://developer.mozilla.org/" target="_blank">doc</a></p> </body> </html>

> Add a CSS sprinkle

 
  @import url(https://overpass-30e2.kxcdn.com/overpass.css);

  body {
    font-family: overpass;
  }

> Write a simple message in JavaScript

 console.log('Hello, World');

5 - start the client server

$ lite-server

If all set, a browser window opens and you should see your html page.

Troubleshooting

- If you end up with the message cannot /GET, check your current location with pwd and ls :
Your current working directory MUST contain an index.html.

6 - start the backend server

$ nodemon

If all set, the message should be printed on your terminal window.

Troubleshooting

- your folder should contain an index.js
- check the terminal for error messages.
- check your files for syntax error.
